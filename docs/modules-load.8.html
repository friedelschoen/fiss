Name -------------------------------------------------------------------------------- modules-load - configure kernel modules at boot Synopsis -------------------------------------------------------------------------------- modules-load [-nv] Description -------------------------------------------------------------------------------- modules-load reads files which contain kernel modules to load during boot from the list of locations below.
  • -n dry-run mode. This option does everything but actually insert or delete the modules.
  • -v verbose mode. Print messages about what the program is doing.
Files -------------------------------------------------------------------------------- Configuration files are read from the following locations:
  • /etc/modules-load.d/*.conf
  • /run/modules-load.d/*.conf
  • /usr/lib/modules-load.d/*.conf
The configuration files should simply contain a list of kernel module names to load, separated by newlines. Empty lines and lines whose first non-whitespace character is # or ; are ignored. History -------------------------------------------------------------------------------- This program is a replacement for the modules-load utility provided by systemd.
