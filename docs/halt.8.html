Name -------------------------------------------------------------------------------- halt, reboot, poweroff - stop the system Synopsis --------------------------------------------------------------------------------
  • poweroff [-n] [-f] [-d] [-w] [-b]
  • reboot [-n] [-f] [-d] [-w] [-b]
  • halt [-n] [-f] [-d] [-w] [-b]
Description -------------------------------------------------------------------------------- poweroff / reboot / halt tells init to stop running services and stop the system. Invoked without -f, it is a shortcut for init <0|6> Options --------------------------------------------------------------------------------
  • -n Don't sync devices, this doensn't imply that the kernel is not synced already
  • -f Forces halt / reboot without notifying init. Forcing the system to halt should be the last hope and can be dangerous, you should not try it
  • -d Don't write the wtmp record
  • w Only write the wtmp record, no further action
